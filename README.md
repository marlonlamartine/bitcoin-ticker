# bitcoin_ticker app
---

A Flutter project following the instructions on the Boss level challenge 3, in the course released by the [londonappbrewery](https://github.com/londonappbrewery).

# Functionality
---

This project is an only screen app, in which the user can select a currency and the app will display its respective value converted into three types of cryptocurrencies: BTC, ETH and LTC. 

In this project we used the [coin api](https://www.coinapi.io/) to fetch our data remotely.

---
